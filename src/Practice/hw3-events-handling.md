# ДЗ 3
Создать новый проект. Проект должен работать следующим образом: Есть два спрайта торговец и рыцарь. Есть два фона: ворота в замок и залы замка. Программа начинается с того что на фоне ворот в замок появляется рыцарь и торговец. При нажатии на рыцаря, тот должен говорить что торговцу нужно самому открыть ворота. При нажатии на ворота торговец проходит в замок, рыцарь исчезает.
### Задача со звездочкой (*)
При нажатии на рыцаря торговец сначала должен спросить у рыцаря как ему пройти в замок, и только после этого рыцарь должен заговорить.
